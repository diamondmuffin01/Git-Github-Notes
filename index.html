Git & Github

Cloning from Github with Git

  • To clone a repo from github we use the clone keyword. Then change to the directory/repo we want to work on. Once we are in the directory we initialize it.



    Next we must create a new branch to push our updated repository. We create a branch with the switch -c followed by the name of the new branch we are creating.

    We can view current branch and other branches with the branch -a command.



    Next we can add/update files



    Finally we push back to the repo.
